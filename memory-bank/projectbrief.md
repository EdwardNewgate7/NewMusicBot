# Project Brief: Harmony Music Bot

## Core Requirements
- A powerful Telegram music bot built with aiogram 3.
- Precise UI matching user-provided screenshots.
- Features: Music/Video playback via Assistant (PyTgCalls), Tagging (Mentions), Scoring system, Playlists, Admin panel.
- High quality (320kbps) audio support.
- Multi-platform media detection and download (Instagram, TikTok, etc.).

## Goals
- Provide a premium user experience with custom emojis and colorful buttons.
- Ensure 24/7 stability for group music sessions.
- Implement robust admin and developer controls.
